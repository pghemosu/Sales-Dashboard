# Sales-Dashboard

📊 Sales Dashboard Overview

This Power BI Sales Dashboard provides a clear and interactive overview of business performance, focusing on key metrics such as total sales (341K) and total profit (27K). It enables users to analyze sales trends over time, compare year-over-year performance, and identify patterns in profitability.

The dashboard breaks down sales by customer segments, product categories, and sub-categories, offering insights into top-performing areas such as Consumer segment and Office Supplies. It also highlights customer behavior through payment modes and operational efficiency via shipping methods.

With interactive regional filtering (Central, East, South, West), users can drill down into location-specific performance, making the dashboard a practical tool for both strategic decision-making and operational analysis.

Designed with clarity and usability in mind, this dashboard demonstrates strong data visualization, storytelling, and analytical skills—making it a solid addition to a data analytics portfolio.

🎯 Purpose
This dashboard is designed to:

1. Monitor overall sales and profit performance
2. Identify trends and seasonal patterns
3. Analyze customer and product segmentation
4. Support strategic business decisions


🛠️ Tools Used:
Power BI – Data visualization and dashboard creation

How to Use
1. Open the Power BI report file.
2. Use the region filter to focus on specific areas.
3. Interact with charts to drill down into insights.
4. Analyze trends and compare year-over-year performance.


Notes:
Data shown is aggregated and may represent sample or historical data.
Dashboard is interactive and supports filtering and cross-highlighting
